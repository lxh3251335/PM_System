# 冷库项目登记管理系统

> **最新版本：V1.1 MVP** - 聚焦核心功能，快速替代Excel表格

## 项目简介

冷库项目登记管理系统是一款面向冷库设备集成商和运营企业的云端管理平台，用于标准化管理冷库项目从立项、设备登记、网关部署到最终上线的全生命周期流程。

### V1.1 MVP核心特性 🆕
- ✅ **项目复制功能**：一键复制历史项目，快速创建相似项目
- ✅ **到货时间管理**：标注货物到货时间，支持筛选和提醒
- ✅ **简化权限**：固定2用户（用户方+厂家方），专注功能本身
- ✅ **平级设备结构**：设备与冷库平级，简化数据结构
- ✅ **快速开发**：预计15天完成，快速上线使用

## 核心功能

- **项目管理**：项目信息登记、邮寄地址管理
- **冷库管理**：冷库信息录入、容积自动计算
- **设备管理**：冷风机、温控器、电表、机组、冷柜等设备登记
- **关系配置**：设备间控制关系建立和可视化
- **网关管理**：网关端口分配、485地址管理
- **流程跟踪**：项目推进流程全记录
- **报表导出**：Excel格式导出各类报表

## 技术栈

### 前端
- Vue 3 + TypeScript + Vite
- Element Plus（UI组件库）
- Vue Router 4（路由管理）
- Pinia（状态管理）
- Axios（HTTP客户端）

### 后端
- FastAPI（Python Web框架）
- SQLAlchemy 2.0（ORM）
- MySQL 8.0（数据库）
- JWT（认证）
- openpyxl（Excel生成）

## 快速开始

### 环境要求

- **Node.js**：16.0+
- **Python**：3.9+
- **MySQL**：8.0+
- **操作系统**：Windows / macOS / Linux

### 开发环境搭建

#### 1. 克隆项目

```bash
git clone <repository-url>
cd PM_System
```

#### 2. 数据库初始化

```sql
CREATE DATABASE pm_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 3. 后端启动

```bash
cd pm-system-backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

访问后端API文档：http://localhost:8000/docs

#### 4. 前端启动

```bash
cd pm-system-frontend
npm install
npm run dev
```

访问前端应用：http://localhost:5173

## 用户角色

### 系统管理员（我方）
- 拥有系统最高权限
- 可管理所有项目和用户
- 可维护标准设备库

### 企业管理员（客户方）
- 仅能管理本企业的项目
- 可使用标准设备库
- 可创建和配置项目

## 主要业务流程

1. **新建项目** → 填写项目基础信息
2. **设备登记** → 创建冷库 → 添加设备
3. **建立关系** → 配置设备间控制关系
4. **网关配置** → 添加网关 → 分配端口和地址
5. **流程推进** → 网关登记 → 邮寄 → 验证 → 上线 → 完成
