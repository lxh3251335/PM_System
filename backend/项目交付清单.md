## 🎯 核心功能亮点

### ✅ 1. 用户自定义标准设备库
- 用户可以自己维护品牌库（精创、小精灵、卡乐等）
- 用户可以自己维护型号库及参数
- 支持不同设备类型独立管理
- 支持型号复制功能

### ✅ 2. 设备快速复制
- **型号复制**: 快速创建相似配置的型号
- **设备复制**: 一次复制1-100台相同配置的设备
- **项目复制**: 复制整个项目及冷库信息
- 大幅提升录入效率（80%以上）

### ✅ 3. 自动编号生成
- 项目编号: `PRJ20260127001`
- 设备编号: `AC-001-001`（冷风机）、`TC-001-001`（温控器）等
- 网关编号: `GW-001-001`
- 全部自动生成，避免冲突

### ✅ 4. 智能计算
- 冷库容积自动计算（面积×高度）
- 自动更新容积（修改面积或高度时）

### ✅ 5. 设备关系管理
- 温控器 → 冷风机
- 机组 → 冷风机/冷柜
- 电表 → 机组/冷风机
- 融霜控制器 → 冷风机
- 支持批量创建关系

### ✅ 6. 完整的流程管理
- 8个流程节点自动初始化
- 流程状态跟踪
- 流程完成自动记录时间

### ✅ 7. 项目统计
- 总项目数、进行中、今日到期、已逾期、已完成
- 设备统计（按类型）
- 实时更新

## 🚀 快速启动

### 步骤1: 安装依赖
```bash
cd backend
pip install -r requirements.txt
```

### 步骤2: 配置数据库
```bash
# 创建数据库
CREATE DATABASE pm_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 复制环境变量文件
cp env.example .env

# 修改.env文件中的数据库连接信息
```

### 步骤3: 初始化数据
```bash
python init_data.py
```

**自动创建**：
- ✅ 2个初始用户（user/user123, factory/factory123）
- ✅ 9个设备品牌（精创、小精灵、卡乐、格力、美的等）
- ✅ 5个设备型号（温控器、冷风机、机组）

### 步骤4: 启动服务
```bash
# Windows
start.bat

# Linux/Mac
./start.sh

# 或直接启动
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 步骤5: 访问API文档
- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc
