# 环境准备指南

## 📋 目标
配置并启动完整的后端服务，为前后端对接做准备。

---

## 第一步：安装MySQL数据库

### Windows 安装步骤

#### 1.1 下载MySQL
访问：https://dev.mysql.com/downloads/mysql/
- 选择：MySQL Community Server 8.0+
- 平台：Windows
- 下载：mysql-installer-community-xxx.msi

#### 1.2 安装MySQL
1. 运行安装程序
2. 选择"Developer Default"（开发者默认）
3. 点击"Execute"安装所有组件
4. 配置MySQL Server：
   - Type: Development Computer
   - Port: 3306（默认）
   - Root密码：**请设置并记住**（例如：root123456）

#### 1.3 验证安装
打开PowerShell或CMD：
```powershell
mysql --version
# 应该显示：mysql  Ver 8.0.xx
```

## 第二步：创建数据库和用户

### 2.1 登录MySQL
```powershell
mysql -u root -p
# 输入刚才设置的root密码
```

### 2.2 创建数据库
```sql
CREATE DATABASE pm_system 
DEFAULT CHARACTER SET utf8mb4 
COLLATE utf8mb4_unicode_ci;
```

### 2.3 创建用户并授权
```sql
CREATE USER 'pm_user'@'localhost' IDENTIFIED BY 'pm_password123';
GRANT ALL PRIVILEGES ON pm_system.* TO 'pm_user'@'localhost';
FLUSH PRIVILEGES;
```

### 2.4 验证
```sql
-- 查看数据库
SHOW DATABASES;

-- 应该能看到 pm_system

-- 退出
exit;
```

## 第三步：配置后端环境

### 3.1 检查Python版本
```powershell
python --version
# 需要：Python 3.8+
```

如果没有Python，下载安装：https://www.python.org/downloads/

### 3.2 进入后端目录
```powershell
cd D:\projects\PM_System\backend
```

### 3.3 创建虚拟环境（推荐）
```powershell
python -m venv venv

# 激活虚拟环境
.\venv\Scripts\activate
```

### 3.4 安装Python依赖
```powershell
pip install -r requirements.txt
```

可能需要的依赖：
- fastapi
- uvicorn
- sqlalchemy
- pymysql
- python-dotenv
- pydantic

### 3.5 配置环境变量
```powershell
# 复制示例配置文件
copy env.example .env
```

编辑`.env`文件（用记事本或VS Code打开）：
```ini
# 数据库配置
DATABASE_URL=mysql+pymysql://pm_user:pm_password123@localhost:3306/pm_system

# 应用配置
APP_NAME=PM_System
APP_VERSION=1.0.0
DEBUG=True

# 服务器配置
SERVER_HOST=0.0.0.0
SERVER_PORT=8000

# CORS配置（允许前端跨域访问）
CORS_ORIGINS=["http://localhost:3000", "http://127.0.0.1:5500", "null"]

# 安全配置
SECRET_KEY=your-secret-key-change-in-production
```

## 🔧 常见问题

### Q1: MySQL服务未启动
**症状：**连接数据库失败
**解决：**
```powershell
# 启动MySQL服务
net start MySQL80
```

### Q2: 端口8000已被占用
**症状：**`Address already in use`
**解决：**
```powershell
# 查找占用端口的进程
netstat -ano | findstr :8000

# 结束进程（PID替换为实际值）
taskkill /PID <PID> /F

# 或者使用其他端口
uvicorn app.main:app --reload --port 8001
```

### Q3: CORS跨域错误
**症状：**浏览器控制台报`CORS policy`错误
**解决：**
检查`.env`文件中的`CORS_ORIGINS`，确保包含：
```ini
CORS_ORIGINS=["http://localhost:3000", "http://127.0.0.1:5500", "null"]
```

### Q4: 数据库连接失败
**症状：**`Access denied for user 'pm_user'@'localhost'`
**解决：**
重新创建用户：
```sql
DROP USER IF EXISTS 'pm_user'@'localhost';
CREATE USER 'pm_user'@'localhost' IDENTIFIED BY 'pm_password123';
GRANT ALL PRIVILEGES ON pm_system.* TO 'pm_user'@'localhost';
FLUSH PRIVILEGES;
```

### Q5: Python依赖安装失败
**症状：**`pip install`报错
**解决：**
```powershell
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
