## 🆘 常见问题排查

### 问题1：页面打开后显示"暂无数据"，但应该有数据

**排查步骤：**
1. 打开浏览器控制台（F12），查看是否有错误
2. 查看Network标签，检查API请求是否成功
3. 检查后端是否正常运行
4. 使用Postman直接调用API：`GET http://localhost:8000/api/projects`

### 问题2：创建项目/设备后，页面没有更新

**排查步骤：**
1. 检查API响应是否成功（200/201状态码）
2. 手动刷新页面（F5）
3. 检查`renderProjects()`等渲染函数是否被调用
4. 查看数据库是否真的保存了数据

### 问题3：CORS错误

**排查步骤：**
1. 确认后端`.env`文件的CORS配置是否包含`"null"`
2. 重启后端服务
3. 清除浏览器缓存
4. 尝试使用Live Server打开前端（`http://127.0.0.1:5500`）

### 问题4：设备关系配置后不显示

**排查步骤：**
1. 检查`device_relations`表是否有数据
2. 检查关系查询API是否正常
3. 检查前端渲染逻辑
4. 验证设备ID是否正确
