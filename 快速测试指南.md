## 🔧 快速切换模式

### 方法一: 修改配置文件

编辑 `demo/assets/js/config.js`:
```javascript
// 演示模式
CONFIG.MODE = 'demo';

// 生产模式
CONFIG.MODE = 'production';
```

### 方法二: URL参数（动态切换）

在URL后面添加参数：
- 演示模式: `project-list.html?mode=demo`
- 生产模式: `project-list.html?mode=production`

## 🐛 常见问题

### Q1: 打开页面报错"api is not defined"

**原因**: 没有正确引入 config.js 和 api.js

**解决**: 确认HTML文件head中有：
```html
<script src="assets/js/config.js"></script>
<script src="assets/js/api.js"></script>
```

### Q2: 生产模式连接不到后端

**症状**: 控制台显示 `net::ERR_CONNECTION_REFUSED`

**解决**:
1. 确认后端已启动：访问 http://localhost:8000
2. 检查config.js中的API地址配置
3. 检查后端CORS配置

### Q3: 数据不显示

**解决**:
1. 按F12打开浏览器控制台
2. 查看是否有JavaScript错误
3. 查看Network标签，检查API请求状态
4. 临时切换到演示模式测试UI

### Q4: SQLite数据库文件在哪里？

**位置**: `D:\projects\PM_System\backend\pm_system.db`

**查看数据**: 
- 使用DB Browser for SQLite打开
- 或使用命令: `sqlite3 pm_system.db .dump`
