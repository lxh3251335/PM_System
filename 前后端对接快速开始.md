# 前后端对接 - 快速开始

## 🎯 目标
快速启动完整的前后端系统，实现真实功能测试。

---

## 📋 前置条件检查

在开始之前，请确认：
- [ ] 已安装Python 3.8+
- [ ] 已安装浏览器（Chrome/Edge推荐）
- [ ] 已有本项目源代码

**注意：** 我们使用**SQLite数据库**，无需安装MySQL！Python自带，零配置！

---

## 🚀 5分钟快速启动

### Step 1: 安装依赖（2分钟）

```powershell
# 进入后端目录
cd D:\projects\PM_System\backend

# 安装Python依赖
pip install -r requirements.txt
```

### Step 2: 配置环境（30秒）

```powershell
# 复制配置文件
copy env.example .env
```

**就这样！无需修改任何配置！** 已默认使用SQLite。

### Step 3: 初始化数据库（1分钟）

```powershell
python init_data.py
```

**预期输出：**
```
开始初始化数据库...
数据库连接成功！
正在创建表结构...
表结构创建成功！
... 更多初始化信息 ...
✓ SQLite数据库文件已创建：pm_system.db
```

### Step 4: 启动后端服务（30秒）

```powershell
# 在backend目录下
start.bat

# 或手动启动
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**看到这个输出就成功了：**
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete.
```

### Step 5: 验证后端（30秒）

浏览器打开以下地址：
1. **API文档**: http://localhost:8000/docs ✅
2. **健康检查**: http://localhost:8000/ ✅
3. **品牌列表**: http://localhost:8000/api/equipment/brands ✅

### Step 6: 启动前端（1分钟）

**方式一：直接打开**
```powershell
# 打开项目列表页
start D:\projects\PM_System\demo\project-list.html
```

**方式二：使用Live Server（推荐）**
1. 用VS Code打开`D:\projects\PM_System\demo`目录
2. 安装"Live Server"插件
3. 右键`project-list.html` → "Open with Live Server"

### Step 7: 开始测试（随时可测）

按照完整流程测试：
1. 打开项目列表页
2. 点击"新建项目"
3. 填写项目信息并提交
4. 创建冷库
5. 登记设备
6. ...完整走一遍

## 🎨 开发模式说明

### 演示模式（Demo Mode）
```javascript
// config.js
CONFIG.MODE = 'demo';
```
- 使用LocalStorage存储数据
- 不需要后端服务
- 适合演示和原型验证

### 生产模式（Production Mode）
```javascript
// config.js
CONFIG.MODE = 'production';
```
- 调用真实后端API
- 数据持久化到MySQL
- 适合真实环境测试

**切换方式：**修改`demo/assets/js/config.js`中的`MODE`值即可。

## 🐛 问题排查

### 前端看不到数据
1. 检查`config.js`的`MODE`是否为`'production'`
2. 按F12打开浏览器控制台，查看错误信息
3. 检查后端是否正常运行（访问http://localhost:8000）
4. 检查网络请求（Network选项卡）

### 后端无法启动
1. 检查端口8000是否被占用
2. 检查MySQL服务是否启动
3. 检查`.env`配置是否正确
4. 查看控制台错误信息

### 数据库连接失败
1. 确认`pm_system.db`文件是否存在
2. 如果不存在，重新执行`python init_data.py`
3. 如果文件损坏，删除后重新初始化
4. SQLite基本不会有连接问题！

## 💡 开发技巧

### 1. 使用浏览器开发者工具
- **Console**: 查看JavaScript错误和日志
- **Network**: 查看API请求和响应
- **Application**: 查看LocalStorage数据

### 2. 使用API文档测试
访问http://localhost:8000/docs，可以直接测试API：
1. 展开任意API端点
2. 点击"Try it out"
3. 填写参数
4. 点击"Execute"
5. 查看响应结果

### 3. 快速切换模式
在`config.js`中添加快捷切换：
```javascript
// 通过URL参数切换模式
const urlParams = new URLSearchParams(window.location.search);
if (urlParams.get('mode')) {
    CONFIG.MODE = urlParams.get('mode');
}
```

使用：
- 演示模式：`project-list.html?mode=demo`
- 生产模式：`project-list.html?mode=production`
