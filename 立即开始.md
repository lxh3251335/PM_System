# 🚀 立即开始 - 3分钟启动系统

## 方式一：一键启动（推荐） ⭐

双击运行：
```
一键启动.bat
```

**就这么简单！** 🎉

脚本会自动：
1. ✅ 检查Python环境
2. ✅ 安装依赖（首次需要2-3分钟）
3. ✅ 初始化数据库
4. ✅ 启动后端服务
5. ✅ 打开前端页面

---

## 方式二：手动启动

### 步骤1: 启动后端（2分钟）

```powershell
cd D:\projects\PM_System\backend
pip install -r requirements.txt
python init_data.py
start.bat
```

### 步骤2: 打开前端（立即）

```powershell
start D:\projects\PM_System\demo\project-list.html
```

## 🐛 遇到问题？

### 问题1: Python未安装
下载安装：https://www.python.org/downloads/

### 问题2: 端口被占用
修改端口：编辑 `backend/.env`

### 问题3: 依赖安装失败
使用国内镜像：
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
