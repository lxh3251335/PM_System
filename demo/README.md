# 冷库项目登记管理系统 - 演示页面使用指南

## 🎉 欢迎使用演示系统！

这是一个完整的冷库项目登记管理系统UI/UX演示，展示了系统的核心功能和界面设计。

---

## 📂 文件结构

```
demo/
├── index.html                 # 落地页（首页）⭐ 从这里开始

├── project-detail.html        # 项目详情页面
├── device-management.html     # 设备管理页面

│   ├── css/

│   │   └── app.css            # 应用样式
│   └── js/

└── README.md                  # 本文件
```

## 🚀 如何浏览

### 方法1：直接打开（推荐）

1. 在文件资源管理器中找到 `demo` 文件夹
2. **双击打开 `index.html`** （这是落地页）
3. 使用浏览器（Chrome、Edge、Firefox）打开

### 方法2：使用VS Code预览

1. 在VS Code中打开 `demo/index.html`
2. 右键 → "Open with Live Server" (需要安装Live Server插件)
3. 或者直接在浏览器中打开文件

### 方法3：使用PowerShell快速打开

```powershell
cd d:\projects\PM_System\demo
start index.html
```

## 📋 页面导航流程

### 推荐浏览顺序：

```
1. index.html (落地页)
   ↓ 点击"免费试用"或"立即体验"
   
2. login.html (登录页)
   ↓ 选择用户类型，点击"立即登录"
   
3. project-list.html (项目列表)
   ↓ 点击"查看"按钮
   
4. project-detail.html (项目详情)
   ↓ 查看5个Tab页：基础信息、冷库、设备、网关、流程
   
5. device-management.html (设备管理)
   ↓ 查看5种设备类型管理
   
6. gateway-config.html (网关配置)
   ↓ 查看端口分配和485地址配置
```

## 🎨 页面功能说明

### 1. **index.html** - 落地页 ⭐

**功能**：
- 产品介绍和价值主张
- 核心功能展示（6大功能）
- 产品优势说明
- 在线演示入口
- 价格方案对比

**亮点**：
- 现代化渐变背景
- 平滑滚动动画
- 3D浏览器模型
- 响应式设计

**操作**：
- 点击导航栏的"在线演示"可查看演示页面链接
- 点击"免费试用"或"立即体验"进入登录页

---

### 2. **login.html** - 登录页

**功能**：
- 双用户类型选择（用户方/厂家方）
- 演示账号自动填充
- 登录验证（模拟）

**演示账号**：
- 用户方：`user001` / `password123`
- 厂家方：`factory001` / `password123`

**操作**：
- 选择用户类型（自动填充用户名）
- 点击"立即登录"跳转到项目列表

---

### 3. **project-list.html** - 项目列表 ⭐

**功能**：
- 项目搜索和筛选
- **到货时间管理**（MVP新功能）
  - 今天到货（橙色）
  - 本周到货
  - 逾期未到（红色）
- 项目复制按钮（MVP新功能）
- 项目CRUD操作

**亮点**：
- 到货时间颜色提示
- 项目流程状态展示
- 操作按钮齐全

**操作**：
- 点击"查看"按钮进入项目详情
- 点击"复制"按钮演示项目复制功能
- 点击"新建项目"弹出创建提示

---

### 4. **project-detail.html** - 项目详情 ⭐

**功能**：
- **5个Tab页**：
  1. 基础信息（含到货时间）
  2. 冷库列表（自动计算容积）
  3. 设备管理（5种设备类型）
  4. 网关配置（链接到网关页面）
  5. **流程进度**（8个流程节点可视化）⭐

**亮点**：
- 流程进度条动画效果
- 清晰的Tab页切换
- 完整的项目信息展示

**操作**：
- 点击Tab切换不同内容
- 查看流程进度的动画效果
- 点击"推进到下一步"演示流程管理

---

### 5. **device-management.html** - 设备管理 ⭐

**功能**：
- **5种设备类型管理**：
  - ❄️ 冷风机 (AC)
  - 🌡️ 温控器 (TC)
  - ⚡ 电表 (PM)
  - 🔌 机组 (UN)
  - 🧊 冷柜 (FR)
- 自动编号展示
- 设备复制功能
- 品牌型号管理

**亮点**：
- 彩色卡片展示设备类型
- 清晰的设备编号规则
- 完整的功能说明

**操作**：
- 查看各种设备类型
- 点击"复制"按钮演示设备复制
- 查看自动生成的设备编号

---

### 6. **gateway-config.html** - 网关配置 ⭐

**功能**：
- 网关信息展示
- **8个端口配置**（可视化卡片）
- 485地址分配
- 配置状态展示（已分配/未分配）

**亮点**：
- 卡片式端口展示
- 颜色区分配置状态
- 清晰的配置说明

**操作**：
- 查看已分配的端口（绿色）
- 查看未分配的端口（灰色）
- 点击"配置"按钮演示端口配置
- 点击"导出配置"演示导出功能

## 🔥 MVP版本核心功能演示

### 1. 项目复制功能 🆕
- **位置**：project-list.html
- **演示**：点击任意项目的"复制"按钮
- **说明**：一键复制项目+冷库+设备，节省80%录入时间

### 2. 到货时间管理 🆕
- **位置**：project-list.html
- **演示**：查看"期望到货时间"列，注意颜色提示
- **说明**：
  - 正常：黑色
  - 今天到货：橙色加粗
  - 逾期未到：红色加粗

### 3. 设备自动编号
- **位置**：device-management.html
- **演示**：查看设备列表的"设备编号"列
- **说明**：格式为 类型前缀-项目ID-序号

### 4. 流程可视化
- **位置**：project-detail.html（Tab 5）
- **演示**：查看8个流程节点的进度条
- **说明**：当前节点有脉冲动画效果

### 5. 网关端口配置
- **位置**：gateway-config.html
- **演示**：查看8个端口的配置卡片
- **说明**：绿色=已分配，灰色=未分配

## 💡 使用技巧

### 查看动画效果
1. 在落地页（index.html）上下滚动
2. 悬停在卡片上查看阴影效果
3. 悬停在按钮上查看动画

### 查看响应式效果
1. 按F12打开浏览器开发者工具
2. 点击"设备工具栏"图标（Ctrl+Shift+M）
3. 选择不同设备查看适配效果

### 查看流程动画
1. 打开 project-detail.html
2. 切换到"流程进度" Tab
3. 观察当前节点的脉冲动画

## 🐛 注意事项

1. **这是纯前端演示**
   - 没有后端API
   - 数据是静态的
   - 按钮点击会弹出提示

2. **功能演示**
   - "登录"、"保存"等按钮只是演示
   - 实际开发时需要连接后端API

3. **浏览器兼容性**
   - 推荐使用Chrome、Edge、Firefox
   - 不支持IE浏览器

4. **本地文件协议**
   - 使用 `file://` 协议打开可能有限制
   - 建议使用Live Server或本地服务器
