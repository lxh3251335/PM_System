# 网关配置页面更新说明 ✅

## 📝 更新内容

### 1️⃣ 修改字段名称
- ❌ **旧**：手机卡号
- ✅ **新**：物联网卡号
- **说明**：更准确地反映实际使用的物联网卡，用于4G/5G网络通讯

### 2️⃣ 端口数自动填充 ⭐ 核心改进
- ❌ **旧**：手动输入"总端口数"
- ✅ **新**：根据品牌和型号自动填充
  - **RS485端口数**：只读显示，自动填充
  - **网口端口数**：只读显示，自动填充

### 3️⃣ 品牌型号联动
- ✅ 选择品牌后，自动加载该品牌的所有型号
- ✅ 型号下拉框显示端口配置（例如：AR151（RS485:8口 网口:2口））
- ✅ 选择型号后，自动填充端口数到对应字段

## 🔧 技术实现

### 网关配置数据结构

```javascript
const gatewayConfig = {
    '华为': {
        'AR151': { rs485: 8, eth: 2, desc: 'AR151' },
        'AR161': { rs485: 16, eth: 4, desc: 'AR161' },
        'AR201': { rs485: 32, eth: 8, desc: 'AR201' }
    },
    '中兴': {
        'ZXR10-5250': { rs485: 8, eth: 2, desc: 'ZXR10-5250' },
        'ZXR10-5260': { rs485: 16, eth: 4, desc: 'ZXR10-5260' }
    },
    '西门子': {
        'SCALANCE-M876': { rs485: 4, eth: 2, desc: 'SCALANCE-M876' },
        'SCALANCE-M826': { rs485: 8, eth: 2, desc: 'SCALANCE-M826' }
    },
    '研华': {
        'EKI-1528': { rs485: 8, eth: 2, desc: 'EKI-1528' },
        'EKI-1532': { rs485: 16, eth: 4, desc: 'EKI-1532' }
    }
};
```

### 交互流程

```
用户操作：
1. 选择品牌 → 触发brandSelect change事件
   ↓
2. 系统加载该品牌的所有型号到型号下拉框
   ↓
3. 用户选择型号 → 触发modelSelect change事件
   ↓
4. 系统自动填充：
   - RS485端口数
   - 网口端口数
```

## 🔄 后续扩展

### 可扩展项
1. **新增品牌**：在`gatewayConfig`对象中添加新品牌及其型号
2. **新增型号**：在对应品牌下添加新型号配置
3. **新增端口类型**：如需要，可添加其他端口类型（如Modbus TCP等）
4. **动态加载**：可从后端API动态加载网关配置数据

### 示例：添加新品牌
```javascript
const gatewayConfig = {
    // ... 现有配置 ...
    '新品牌': {
        '型号A': { rs485: 12, eth: 4, desc: '型号A' },
        '型号B': { rs485: 24, eth: 8, desc: '型号B' }
    }
};
```
